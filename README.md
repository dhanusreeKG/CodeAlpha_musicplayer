A GTA neon-pink themed music player built with HTML, CSS & JavaScript. Features play, pause, next, previous, volume control, progress bar, playlist, and autoplay.  

## How to Use

1. Clone or download the repo.  
2. Keep index.html and your .mp3 files in the same folder.  
3. Open index.html in a browser to play music.  

## Add More Songs

- Place your MP3 in the folder.  
- Update the songs array in index.html with new song details.
